# Modélisation HMM et régression angulaire sur  les trajectoires de fourmis

Ce dépôt contient les scripts R utilisés dans mon essai de maîtrise :
- `code/pretraitement_et_exploration_des_donnees.R` : nettoyage et figures descriptives
- `code/Hmm_essai.R` : ajustement des modèles HMM et production des sorties
- `code/regression_angulaire.R` : régression angulaire de type consensus AR(p)

Les scripts sont organisés par chapitre de l’essai et permettent de
reproduire les principales analyses et figures du manuscrit.
